## Summary

Create Jupyter notebook versions of demos 01 (`01_core_apis`) and 02 (`02_kmeans`) that provide an interactive, cell-by-cell learning experience. The notebooks call the existing `run_demo()`, `run_cpu_baseline()`, and `run_gpu_version()` functions — they are presentation wrappers, not reimplementations.

---

## Background

ML practitioners and researchers frequently use Jupyter notebooks as their primary learning and exploration environment. The current demos are Python scripts runnable from the CLI, which is appropriate for production use but less interactive for learners who want to modify parameters and observe effects incrementally. Demos 01 and 02 are the most foundational (device setup + first GPU kernel, and the first ML algorithm), making them the highest-value first notebooks to deliver.

The `notebooks/` directory is already defined in the project structure (CLAUDE.md) and currently empty.

## API / Interface Design

```python
# notebooks/01_core_apis.ipynb — Cell 4: Parameter setup (user-editable)
N_SAMPLES = 1_000_000
SEED = 42
BLOCK_SIZE = 256   # Try: 64, 128, 256, 512

# notebooks/01_core_apis.ipynb — Cell 6: CPU baseline
from demos.demo_01_core_apis import run_cpu_baseline
import numpy as np
data = np.random.default_rng(SEED).random(N_SAMPLES, dtype=np.float32)
cpu_result, cpu_time = run_cpu_baseline(data)
print(f"CPU time: {cpu_time*1000:.2f} ms")

# notebooks/01_core_apis.ipynb — Cell 7: GPU version (guarded)
if GPU_AVAILABLE:
    from demos.demo_01_core_apis import run_gpu_version
    gpu_result, gpu_time = run_gpu_version(data, block_size=BLOCK_SIZE)
    print(f"GPU time: {gpu_time*1000:.2f} ms  |  Speedup: {cpu_time/gpu_time:.1f}x")
else:
    print("GPU not available — skipping GPU cell")
```
