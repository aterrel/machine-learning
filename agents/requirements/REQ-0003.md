## Summary

Teach users how to write CUDA C kernel code inline in Python strings, compile them with NVRTC via `cuda.core.Program`, and launch them for ML-relevant operations: matrix multiply, softmax, ReLU, and element-wise fused operations.

---

## Background

The most powerful feature of CUDA Python is the ability to write arbitrary CUDA C kernels from Python without a separate C++ compilation step. ML practitioners who want to implement novel operations (custom loss functions, fused kernels) need to know this workflow. This demo covers the full authoring lifecycle: write → compile → launch → verify.

## Notes

- Tiled GEMM with shared memory is the canonical example of GPU memory hierarchy usage
- Fused kernels reduce memory round-trips — key for ML inference performance
