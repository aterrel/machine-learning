## Key Implementation Decisions

1. **`cuda.core.experimental`** used throughout (cuda-python >= 12.3 API).
2. **Centroid initialisation uses same seed/RNG sequence** in both CPU and GPU variants to ensure comparable results.
3. **Empty cluster handling** in GPU k-means falls back to a CPU-side random sample reinitialisation after D2H copy of counts.
4. **DeviceBuffer.close()** called via `__exit__` to ensure no GPU memory leaks.
5. **`extern "C"`** on all CUDA kernels to avoid NVRTC name mangling.
6. **Pinned memory** allocated via `cudaHostAlloc` / `cudaFreeHost`; not GC-managed.
7. **CUDA event timing** in `BenchmarkRunner.time_gpu()` using `cudaEventElapsedTime`.
